## **How to Use the Baldur’s Gate 3 Save Editor**

1. Download **BG3-Save-Editor** from the [Releases](#) page  
2. Launch the app (no installation needed)  
3. Locate your *Baldur’s Gate 3* save folder  
4. Select the save file you want to edit  
5. Adjust character stats, quests, or world settings  
6. Click **Save Changes** to apply edits

## ⚠️ **User Warnings**

### **Protecting Your Save Files**

Modifying save files carries some risks, including:

- Broken or incomplete quest progress  
- Companion synchronization errors  
- Game crashes, freezes, or unexpected behavior  
- Permanent save file corruption  

**Always create backups before editing your saves.**  

---

## **Automatic Backup System**

Before any edits are applied, the editor generates safe, timestamped backups:

- Full copy of the original save file  
- Restore points with `SaveName_yyyyMMdd_HHmmss.bak` format  
- Backups saved in a `Backups` folder next to the original save  
- Automatic cleanup keeps only the 5 most recent backups  

This ensures you can **quickly restore any save** if an edit causes problems, keeping your *Baldur’s Gate 3* experience safe and worry-free.
